## Example
<Snippet file={file} title={'Checkbox 1'} preview={true}>
  <Fragment slot="html">
    ```
    <div class="form">
      <div class="field">

          <input class="checkbox" type="checkbox" id="input1" value=""/>

        <div class="inline">
          <input class="checkbox" type="checkbox" id="input2" checked/>

      </div>
    </div>
    ```

    ```
      'form',

      app.$theme
    );
    ```
