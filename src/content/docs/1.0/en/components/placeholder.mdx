## About

Placeholders can be used to enhance the experience of your application. They're built only with HTML and CSS, meaning you don't need any JavaScript to create them. You will, however, need some custom JavaScript to toggle their visibility. Their appearance, color, and sizing can be easily customized with our utility classes.

## Example

In the example below, we take a typical card component and recreate it with placeholders applied to create a "loading card". Size and proportions are the same between the two.
<Snippet file={file} title={'Placeholder 1'} preview={true}>
    <Fragment slot="html">
        ```
        <div class="placeholder" aria-hidden="true">
            <span class="el"></span>
            <span class="el"></span>
            <span class="el"></span>
            <span class="el"></span>
            <span class="el"></span>
        </div>
        ```
    </Fragment>
    <Fragment slot="scss">
        ```
        @use '~/styles/app.scss' as app;

        $placeholder: app.get-theme(app.$theme, 'components.placeholder');

        @include app.generate-component(
            $placeholder,
            'placeholder',
            app.$config,
            app.$theme
        );
        ```
    </Fragment>
</Snippet>

### Width

You can change the `width` through grid column classes, width utilities, or inline styles.

<Snippet file={file} title={'Placeholder 2'} preview={true}>
    <Fragment slot="html">
        ```
        <div class="placeholder" aria-hidden="true">
            <span class="el"></span>
            <span class="el"></span>
            <span class="el" style="width: 25%;"></span>
        </div>
        ```
    </Fragment>
